# Cabal

Initialize a new project interactively:

    $ cabal init --interactive

Open a GHCi REPL:

    $ cabal repl

Run an executable:

    $ cabal run EXECUTABLE

Run the tests:

    $ cabal test

Install a dependency:

    $ cabal install PACKAGE

Install a library dependency:

    $ cabal install --lib LIBRARY

Cleanup the project:

    $ cabal clean

Build the project:

    $ cabal build
